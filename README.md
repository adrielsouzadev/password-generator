# 🔐 Password Generator

Gerador de senhas seguras desenvolvido com HTML, CSS e JavaScript.

## 🚀 Demonstração

Acesse o projeto online:

https://adrielsouzadev.github.io/password-generator/

## ✨ Funcionalidades

- Geração de senhas aleatórias
- Definição do tamanho da senha
- Inclusão de letras maiúsculas
- Inclusão de letras minúsculas
- Inclusão de números
- Inclusão de símbolos
- Indicador de força da senha
- Mostrar/Ocultar senha
- Copiar senha para a área de transferência
- Geração segura utilizando `crypto.getRandomValues()`

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 💻 Como Executar Localmente

Clone o repositório:

```bash
git clone https://github.com/adrielsouzadev/password-generator.git
```

Entre na pasta:

```bash
cd password-generator
```

Abra o arquivo `index.html` no navegador.
